== Description ==
Use this calculator to work out the approximate monthly repayments you will need to make on a personal loan or mortgage. Free  Loan Calculator for your website.

To use Loan Calculator on your site, please put shortcode [loancalculator] in any page.


- You can change a rate of interest from setting page (default 12.50%).
- You can change a heading of block from setting page (default Loan Calculator).
- You can change a file charges from setting page (default 325$).



== Installation ==
Using  Wordpress admin
 - Go to plugin menu.
- Add new plugin and upload plugin.
- Activate plugin form list.

Using FTP

-     Upload loan-calculator-maczin to the /wp-content/plugins/ directory
-     Activate the plugin through the ‘Plugins’ menu in WordPress.
